# Pet Shelter

A web application for managing pet adoptions and shelter operations.

## Features

- List available pets
- Add, edit, and remove pet profiles
- Adoption application management

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/pet-shelter.git
cd pet-shelter
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```
